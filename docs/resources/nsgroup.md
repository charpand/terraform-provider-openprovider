# openprovider_nsgroup (Resource)

Manages an OpenProvider nameserver group. Nameserver groups can be referenced by domains using the `ns_group` field.

## Example Usage

### Basic

```terraform
resource "openprovider_nsgroup" "example" {
  name = "my-ns-group"

  nameservers {
    name = "ns1.example.com"
  }

  nameservers {
    name = "ns2.example.com"
  }
}
```

### With IP Addresses

```terraform
resource "openprovider_nsgroup" "with_ips" {
  name = "my-ns-group-with-ips"

  nameservers {
    name = "ns1.example.com"
    ip   = "192.0.2.1"
    ip6  = "2001:db8::1"
  }

  nameservers {
    name = "ns2.example.com"
    ip   = "192.0.2.2"
    ip6  = "2001:db8::2"
  }
}
```
